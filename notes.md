# Shell Scripting Basics

## What is a Shell Script?

Text files that contain commands to be executed by a shell are called **shell scripts**.

Shell scripting allows users to automate repetitive tasks by combining multiple commands into a single file.

---

## Key Concepts

### Shell

A **shell** is a program that acts as an interface between the user and the operating system's kernel. It interprets human-readable commands into instructions the OS can understand.

Common shells include:
- Bash (Bourne Again Shell)
- Zsh
- Bourne Shell (sh)

---

### Script

A **script** is a plain text file containing:
- Commands
- Control flow structures (loops, conditionals)
- Variables
